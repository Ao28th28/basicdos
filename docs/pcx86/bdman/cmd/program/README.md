BASIC-DOS programs consist of:

- [Commands](../)
- [Statements](#statements)
- [Functions](#functions)

### Statements

- [DEF](def/) *function*
- [GOTO](goto/) *line*
- [IF](if/) *expression* THEN *statement(s)*
- [LET](let/) *variable* = *expression*
- [RETURN](return/) [*expression*]

- RND%

### Variables
